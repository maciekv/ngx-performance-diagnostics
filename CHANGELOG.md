# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-12-10

### Added
- Initial release
- Change Detection Monitor directive
- Change Detection Monitor service
- Memory Leak Detector service
- CD Monitor Panel component
- Memory Leak Panel component
- Comprehensive documentation
- TypeScript support with strict mode
- Angular 17+ support
- Standalone components support

### Features
- Real-time change detection monitoring
- Memory leak detection
- Visual diagnostic panels
- Export reports to JSON
- DOM element highlighting
- Memory trend analysis
- Garbage collection forcing
- Zero configuration setup
